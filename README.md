## Installation

### Install CLIs
```bash
# Copy to your bin directory
cp cli/aria ~/bin/
cp cli/lucidia ~/bin/
chmod +x ~/bin/aria ~/bin/lucidia

# Add to PATH (add to ~/.zshrc or ~/.bashrc)
export PATH="$HOME/bin:$PATH"

# Use the CLIs
aria --status
lucidia --specialties
```

### Deploy Website
```bash
# Deploy to Cloudflare Pages
cd website
wrangler pages deploy . --project-name=aria-blackroad-me
```
